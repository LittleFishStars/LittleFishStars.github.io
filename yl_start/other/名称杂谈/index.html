空

名称杂谈

  1. 月落星沉 月落

    皓月当空，亦有落时；

    繁星虽众，亦有沉日；

    日月变换，唯吾心永恒。

  2. 灵空

    因灵而空，谓之空灵

    灵由空生，谓之灵空

  3. 夏梦

    浮生几何，不过夏日一梦，梦醒之时，一切皆虚无。

  4. 苏兰

    兰草虽凡，于无声处惊艳，恍若流苏般华美，再寻无门。

  5. 秦武 秦老

    秦时月，汉时关，长夜之下，唯有守护。

发布于 2023.6.30
